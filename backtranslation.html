2025-09-17 • status: in-progress • importance: medium
Creating good training data for RL on verifiable rewards does not require in-depth understanding of the data. At minimum, it requires taking existing "answers" and fitting them with appropriate questions. Consider that a model doesn't have to be able to get questions on coding problems correct to be able to take a textbook with given answers and simply format it into appropriate JSON for training. In a 2023 paper from Meta, researchers discuss using a similar technique to create instruction-tuning data for language models. Taking the mass of data on the internet and converting answers to given questions on QA forums and similar, a model can simply convert that to the proper "prompt : answer" pair format required for instruction-tuning, and additionally come up with questions to befit answers.
